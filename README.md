# 🌻 植物大戰殭屍網頁版

一個使用純 JavaScript、HTML 和 CSS 開發的植物大戰殭屍風格塔防遊戲。

## 🎮 遊戲玩法

### 目標
在你的草坪上種植植物來防禦不斷進攻的殭屍大軍!

### 遊戲規則
1. **收集陽光** - 點擊從天空掉落的陽光來收集資源
2. **種植植物** - 使用陽光購買並放置植物
3. **防禦殭屍** - 阻止殭屍到達房屋左側
4. **贏得勝利** - 擊敗所有 5 波殭屍

### 植物類型

| 植物 | 成本 | 功能 |
|------|------|------|
| 🌻 向日葵 | 50 陽光 | 每 10 秒產生 25 陽光 |
| 🌱 豌豆射手 | 100 陽光 | 發射豌豆攻擊殭屍 (傷害: 20) |
| 🛡️ 堅果牆 | 50 陽光 | 高血量防禦 (HP: 300) |
| 💣 櫻桃炸彈 | 150 陽光 | 3秒後爆炸 (範圍傷害: 180) |

### 殭屍類型
- **普通殭屍**: HP 100, 移動緩慢, 傷害 10/秒

- **滑鼠左鍵**: 點擊陽光收集
- **ESC鍵**: 取消選擇的植物

## 🎲 遊戲機制

### 波次系統
- 第 1 波: 10 秒後開始
- 之後每 30 秒一波
- 共 5 波殭屍
- 難度逐漸增加

### 陽光系統
- 初始陽光: 50
- 天空陽光: 每 10-20 秒隨機掉落
- 向日葵生產: 每 10 秒產生 25 陽光

### 勝利/失敗條件
- **勝利**: 擊敗所有 5 波殭屍
- **失敗**: 任何殭屍到達草坪左側
